# AI LMS with AI Tutor 🚀

This is a full-stack Learning Management System (LMS) with an AI-powered tutor.

## 🔧 Tech Stack

* Node.js
* Express.js
* MongoDB (Connected ✅)
* OpenAI API (for AI tutor)


Then open:
http://localhost:5000
